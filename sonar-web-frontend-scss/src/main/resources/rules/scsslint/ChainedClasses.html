Details

Disabled by default

Reports when you define a rule set using a selector with chained classes (a.k.a. adjoining classes).

Bad

.foo {
  padding: 5px;
}

.bar {
  margin: 5px;
}

.foo.bar {
  display: block;
}

Good: write chained classes as new class

.foo {
  padding: 5px;
}

.bar {
  margin: 5px;
}

.new-class {
  display: block;
}
